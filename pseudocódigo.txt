INICIAR lista_de_tareas COMO lista vazia
INICIAR contador_id COMO 1

ENQUANTO verdadeiro FAÇA 
   MOSTRAR menu: 
1- Adicionar tarefa
2- VER tarefas
3- Concluir tarefa
4- Remover tarefa 
5- Sair

LER opção do usuário

SE opçaõ == 1:
LER titulo da tarefa 
CRIAR dicionário tarefa com id, titulo e status "pendente"
ADICIONAR tarefa á lista_de_tarefas
INCREMENTAR contador_id

SENÂO SE opção == 2
PARA cada tarefa em lista_de_tarefas:
  MOSTRAR id, titulo e status

SENÃO SE opção == 3 
LER id da tarefa a concluir
PARA cada tarefa em lista_de-tarefas:
 SE id == id informado:
   MUDAR status para "concluida"

SENÃO SE opção == 4
LER id da tarefa a remover
REMOVER tarefa co id correspondente

SENÃO SE opção == 5 
ENCERRAR programa

SENÃO: 
   MOATRA "opção inválida"
